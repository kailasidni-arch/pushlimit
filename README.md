# 🏋️‍♂️ FitLife Gym - Web Management System

Sebuah aplikasi web manajemen gym modern yang dibangun menggunakan PHP untuk menangani dinamika konten dan fungsionalitas server-side. Project ini dirancang dengan struktur folder yang rapi dan antarmuka pengguna (UI) yang agresif serta energetik.

## 🚀 Live Demo
Web ini di-hosting menggunakan InfinityFree. Kamu bisa melihat tampilan live-nya di sini:
👉 **[Link InfinityFree Kamu di Sini]**

## ✨ Fitur Utama
- **Dynamic Content:** Menggunakan PHP untuk pengelolaan data dan halaman yang lebih fleksibel.
- **Modern UI/UX:** Desain bertema gelap (Dark Mode) dengan aksen warna yang kontras untuk kesan sporty.
- **Responsive Layout:** Tampilan yang menyesuaikan di berbagai perangkat (Mobile, Tablet, Desktop).
- **Membership Management:** Halaman khusus untuk menampilkan paket langganan dan kelas fitness.
- **Organized Structure:** File tertata rapi dalam folder (assets, admin, dll) untuk kemudahan maintenance.

## 🛠️ Tech Stack
- **Backend:** PHP
- **Frontend:** HTML5, CSS3, JavaScript
- **Hosting:** InfinityFree (Apache Server)

## 📁 Struktur Folder
Project ini memiliki struktur yang terorganisir untuk memisahkan logika koding dan aset visual:
- `/admin` : Berisi file manajemen untuk admin gym.
- `/assets` : Berisi file CSS, gambar/images, dan script JS.
- `index.php` : Halaman utama/landing page.

## 📝 Instalasi Lokal
Jika ingin menjalankan project ini di komputer lokal:
1. Pastikan sudah menginstall XAMPP/Laragon.
2. Clone repository ini ke folder `htdocs`.
3. Jalankan Apache dari Control Panel XAMPP.
4. Akses melalui browser di `localhost/gym-website-project`.
